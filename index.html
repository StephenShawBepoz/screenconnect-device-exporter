Features

Everything you need

Built for IT teams who manage device fleets through ScreenConnect.

Instant Export

Single API call fetches all devices in seconds. No clicking through one by one.

Smart Enrichment

Auto-detects OS end-of-life dates, processor age, and splits state from location fields.

Name Search

Filter devices by name — type "Kickin" to export only matching devices.

Any Instance

Works on screenconnect.com, self-hosted, and custom domains. No API keys needed.

CSV or JSON

Spreadsheet-ready CSV with clean headers, or structured JSON for automation.

Session Filtering

Filter by Access, Support, or Meeting. Shows device count after each export.
