ScreenConnect Device Exporter

Export device details from any ScreenConnect instance to a spreadsheet in one click. Powered by the Report API — fetches all devices instantly instead of clicking through them one by one.

Works with Google Chrome and Microsoft Edge (any Chromium browser).

How to Use

Once installed, exporting takes seconds — no scrolling or waiting required.

1

Open ScreenConnect

Go to any page in your ScreenConnect instance and make sure you're logged in.

2

Click the extension and hit Export

Click the Device Exporter icon in your toolbar. Choose CSV or JSON, pick a session type (defaults to Access), then click Export CSV.

The extension fetches all devices via the Report API in one request — no clicking through devices one by one.

3

Open the downloaded file

A file will download automatically within seconds. Open it in Excel, Google Sheets, or Numbers to see all your device data.

Requires Report Manager. This extension uses the built-in ScreenConnect Report API. If you get a "Report API not found" error, the Report Manager extension may need to be enabled in your ScreenConnect Admin > Extensions.
Some fields are blank? This is normal. Devices that are offline or have an older ScreenConnect agent may not report all hardware details (like RAM or Processor). The extension exports whatever ScreenConnect has stored.
Works on any instance. This extension is not limited to screenconnect.com — it works on self-hosted and custom-domain ScreenConnect instances too. Just open your instance, log in, and click Export.
